StyleShifter scans your CSS for special expression markers like /*![demo.primaryColor]*/ and generates scoped overrides on-the-fly. No build process, no CSS preprocessors - just pure runtime magic.

Expression Syntax

StyleShifter works beautifully with CSS custom properties. Define variables once, use them everywhere, and update them at any level with StyleShifter expressions.

/* Define CSS variables */ :root { /*![demo.primaryButton]*/ --color-primary: #667eea; /*![demo.accentColor]*/ --color-accent: #ed8936; } /* Use variables throughout your CSS */ .button { background: var(--color-primary); border-color: var(--color-accent); }
