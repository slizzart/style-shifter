üìò How It Works

StyleShifter scans your CSS for special expression markers like /*![demo.primaryColor]*/ and generates scoped overrides on-the-fly. No build process, no CSS preprocessors - just pure runtime magic.

Expression Syntax

/*![theme.propertyName]*/ background-color: #defaultValue; /*![functionName(theme.property, args)]*/ color: rgba(0, 0, 0, 1);
